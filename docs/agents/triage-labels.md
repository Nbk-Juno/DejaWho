# Triage Labels

The shared triage workflow uses these canonical roles:

| Role | GitHub label |
| --- | --- |
| Needs maintainer evaluation | `needs-triage` |
| Waiting on reporter or more context | `needs-info` |
| Fully specified and ready for an agent | `ready-for-agent` |
| Ready for human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

Use these exact labels when creating, updating, or triaging GitHub Issues.
